## 📋 Test Case & Test Plan Samples

### Test Case: Testing “Payment” Form in E-Commerce Application
- **Test Case ID:** GROW-177  
- **Description:** Testing the “Payment” Form with valid credentials  
- **Steps:**  
  1. Access the checkout page with items in the cart.
  2. Click the "Credit Card" payment option.
  3. Click "Place Order".
  4. Enter valid card details.
  5. Click "Pay Now".

- **Expected Result:** "Order Successful" popup should appear for valid input.  
- **Actual Result:** "Order Successful" popup did NOT appear for valid input.  
- **Status:** Fail
